# YelpHelper

##To run Client app 
  * cd client
  * npm install
  * npm start
  
Client app runs on http://localhost:3000
 
##To run Server app
 * cd server
 * npm install
 * nodemon
 
Server app runs on http://localhost:5000

## Packages Used in App
* node :  https://nodejs.org/en/download/
* react
* redux
* express
* cors
* axios : https://github.com/axios/axios
* lodash : https://lodash.com/docs/4.17.10
